# Python Maze Solver

#### Usage

```sh
# Create virtual environment
python3 -m venv .venv
# Activate virtual environment
. ./.venv/bin/activate
```

On `Linux`:
```sh 
# Run script and pass maze file and type:
./main.py -f ./lab_3.txt -t 2

# In case pygame module can't be loaded:
python3 main.py -f ./lab_1.txt -t 1
```
